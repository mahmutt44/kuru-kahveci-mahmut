\# Kuru Kahveci Mahmut ☕



Flask + SQLite tabanlı kuru kahveci satış uygulaması.



\## Özellikler

\- Ürün listeleme ve detay

\- Sepet ve sipariş sistemi

\- Stok kontrolü

\- Admin panel (ürün \& sipariş yönetimi)

\- Telefon numarası ile sipariş sorgulama

\- Bootstrap 5 arayüz



\## Kurulum

```bash

pip install -r requirements.txt

python app.py
